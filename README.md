# Friendly Project

An Android application project with boilerplate configuration and structure.

## Project Structure

- `app/` - Main Android application module
- `build.gradle` - Root-level Gradle configuration
- `settings.gradle` - Gradle settings and module definitions
- `.gitignore` - Git ignore file for Android projects

## Building

To build the project:

```bash
./gradlew build
```

To run the app:

```bash
./gradlew installDebug
```

## Dependencies

- AndroidX AppCompat
- Material Components
- ConstraintLayout
- JUnit (testing)
- Espresso (UI testing)
